    WindowsからRaspberry Piにリモートデスクトップ接続

    Raspberry Pi環境構築

    sudo apt-get update
sudo apt-get install tightvncserver -y
sudo apt-get install xrdp

    Windows リモートデスクトップを起動し、IPを指定。

    USER:pi
PW(初期):raspberry

    ラズパイ opencv 手軽環境構築

    wget https://github.com/mt08xx/files/raw/master/opencv-rpi/libopencv3_3.4.0-20180115.1_armhf.deb
sudo apt install -y ./libopencv3_3.4.0-20180115.1_armhf.deb
sudo ldconfig

    opencv導入の確認

    import cv2

    USBカメラ認識の確認。以下のコードをpythonで実行

    import numpy as np
import cv2

cap = cv2.VideoCapture(0)
while True:
    imgs = []
    ret, img = cap.read()
    cv2.imshow('capture', img)
    ch = cv2.waitKey(1)
    if ch == 27:
        break
cv2.destroyAllWindows()

    ラズパイWebカメラstreaming配信(M-JPEG streamer)

    環境構築

    cd ~
mkdir mjpg-streamer
cd mjpg-streamer
sudo apt-get install subversion libjpeg-dev imagemagick
svn co https://svn.code.sf.net/p/mjpg-streamer/code/mjpg-streamer mjpg-streamer
cd mjpg-streamer/mjpg-streamer
make
sudo make install

    mjpeg-streamer ラズパイサーバーの起動

    #!/bin/sh

PORT="8090"
SIZE="640x480"
FRAMERATE="20"

cd /home/pi/mjpg-streamer/mjpg-streamer
sudo ./mjpg_streamer -i "./input_uvc.so -f $FRAMERATE -r $SIZE -d /dev/video0 -y -n" -o "./output_http.so -w ./www -p $PORT"

    http://(ラズパイのip):(上記で指定したポート)でアクセスするとブラウザで閲覧可
